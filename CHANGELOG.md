# Changelog

All notable changes to Chrome Clean are documented here.

## 1.0.0 - 2026-07-14

- Limited cleanup to Chrome and Chromium policy registry keys.
- Removed deletion of system-wide Windows Group Policy directories.
- Removed deletion of general Chrome application and enrollment data.
- Added a policy preview and explicit confirmation step.
- Added automatic registry backups before deletion.
- Added accurate success, cancellation, and failure exit codes.
- Aligned the README with the repository's GPLv3 license.
- Expanded safety, authorization, recovery, and troubleshooting guidance.
